The Biogas Monitoring System is an advanced ESP32-based IoT solution designed for real-time monitoring of biogas storage tanks. This system helps users monitor gas levels, temperature, and tank status remotely through WiFi connectivity and cloud integration.

The project includes a modern WiFi configuration portal, Google Sheets data logging, email alert system, and smart sensor monitoring using ultrasonic and DS18B20 temperature sensors. Users can configure system settings directly from a web portal without re-uploading firmware.

Key Features:
✔ Real-Time Biogas Level Monitoring
✔ ESP32 WiFi Configuration Portal
✔ Google Sheets Cloud Logging
✔ Email Notification Alerts
✔ DS18B20 Temperature Monitoring
✔ Ultrasonic Gas Level Detection
✔ Adjustable Upload Interval
✔ OTA Firmware Ready
✔ Mobile Friendly Web Interface
✔ One-Click Firmware Installation

This project is designed for smart agriculture, renewable energy systems, research projects, and industrial biogas monitoring applications.

The firmware and web interface are fully customizable and open for future upgrades including OTA updates, dashboard analytics, and mobile app integration.

Designed with modern IoT technology to make biogas monitoring smarter, easier, and more efficient.
